🧪 DOM Manipulation Practice

A simple guide for testing webpage changes with Inspect Element

🧩 What is DOM Manipulation?

The DOM is the browser's live version of a webpage. It includes the page title, headings, paragraphs, buttons, colors, and styles.

DOM manipulation means using JavaScript to change something on the page after it loads. For example, you can change text, colors, button labels, or the page title.

Teacher note: Changes made in Inspect Element or the Console only affect your screen. Refreshing the page brings the original version back.

🔍 How to Test the Commands

  1. Open this page in a browser like Chrome, Edge, or Firefox.
  2. Right-click anywhere on the page and choose Inspect or Inspect Element.
  3. Click the Console tab in the panel that opens.
  4. Copy one command below, paste it into the Console, and press Enter.
  5. Refresh the page when you want to undo your changes.

⌨️ Commands to Try

Each command starts with document, which means JavaScript is talking to the webpage document.

Change all text color

document.body.style.color = "blue";

Makes the text on the page blue.

Change the browser tab title

document.title = "My New Title";

Changes the title shown in the browser tab.

Change the first heading

document.querySelector("h1").textContent = "My New Title";

Finds the first h1 heading and replaces its text.

Make the page text huge

document.body.style.fontSize = "100px";

Makes the page text very large. Refresh to reset it.

Change the page background

document.body.style.background = "red";

Changes the background behind the page to red.

Change the first button

document.querySelector("button").textContent = "Click Me!";

Finds the first button and changes the words inside it.

🎯 Practice Area Commands

These commands are better for practice because they target only the box with the class practice-area.

Notice the period before the class name. In CSS and JavaScript selectors, .practice-area means "find something with this class."

Change only the practice box text color

document.querySelector(".practice-area").style.color = "blue";

Only the text inside the practice area turns blue.

Change only the practice box background

document.querySelector(".practice-area").style.background = "red";

Only the practice box background turns red.

Change the practice box heading

document.querySelector(".practice-area h2").textContent = "My New Title";

Finds the h2 inside the practice area and changes its words.

Change the practice box button

document.querySelector(".practice-area button").textContent = "Click Me!";

Finds the button inside the practice area and changes its label.

🧠 What These Commands Have in Common

  • document means the webpage loaded in the browser.
  • body means the visible page area.
  • style means visual settings like color, size, or background.
  • querySelector finds the first matching element, such as h1 or button.
  • .practice-area finds an element with the class practice-area.
  • textContent changes the text inside an element.
document.querySelector(".practice-area h2").textContent = "My New Title";

In plain English: Find the heading inside the practice area and change its words to "My New Title."
